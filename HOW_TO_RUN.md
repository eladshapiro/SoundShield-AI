# איך להריץ את המערכת / How to Run the System

## ✅ המערכת מותקנת ועובדת! / System is Installed and Working!

### 🚀 דרכים להרצת המערכת / Ways to Run the System:

#### 1. **דוגמה מהירה (הכי קל) / Quick Example (Easiest)**
```bash
py example_usage.py
```
זה יוצר קובץ אודיו לדוגמה ומנתח אותו אוטומטית.

#### 2. **ממשק ווב (מומלץ) / Web Interface (Recommended)**
```bash
py web_app.py
```
ואז פתח בדפדפן: **http://localhost:5000**

#### 3. **שורת פקודה / Command Line**
```bash
py main.py your_recording.wav
```

### 🎯 איך להשתמש / How to Use:

#### עם קבצי אודיו אמיתיים / With Real Audio Files:
1. העלה קובץ אודיו (WAV, MP3, M4A, FLAC, AAC, OGG)
2. המערכת תנתח ותזהה:
   - **רגשות מדאיגים** - כעס, לחץ, אגרסיה
   - **בכי תינוקות** - זיהוי ובחינת תגובת הצוות
   - **אלימות** - צעקות, איומים, אלימות מילולית
   - **הזנחה** - אי תגובה לבכי וחוסר אינטראקציה

#### פורמטי אודיו נתמכים / Supported Audio Formats:
- WAV, MP3, M4A, FLAC, AAC, OGG
- עד 500MB לקובץ
- כל קצב דגימה נתמך

### 📈 רמות חומרה / Severity Levels:
