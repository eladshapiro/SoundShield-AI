# Kindergarten Recording Analyzer
## מערכת ניתוח הקלטות גן ילדים

מערכת מתקדמת לניתוח הקלטות אודיו מהגן לזיהוי התנהגויות לא מתאימות של הצוות.

### Features / תכונות:
- **זיהוי רגשות** - זיהוי כעס, לחץ, אגרסיה בקול הצוות
- **זיהוי בכי תינוקות** - זיהוי בכי ובדיקת תגובת הצוות
- **זיהוי הזנחה** - זיהוי אי תגובה לבכי וחוסר אינטראקציה
- **זיהוי אלימות** - זיהוי צעקות, איומים ואלימות מילולית
- **דוחות מפורטים** - דוחות בעברית ובאנגלית עם חותמות זמן
- **ממשק ווב** - ממשק ידידותי להעלאת קבצים וצפייה בתוצאות

### Installation / התקנה:

#### 1. התקנת תלויות / Install Dependencies:
```bash
pip install -r requirements.txt
```

#### 2. יצירת תיקיות נדרשות / Create Required Directories:
```bash
mkdir uploads reports models
```

### Usage / שימוש:

#### 1. הרצה משורת הפקודה / Command Line Usage:
```bash
# ניתוח קובץ יחיד
python main.py recording.wav

# הרצת דוגמה
python example_usage.py
```

#### 2. הרצת ממשק ווב / Web Interface:
```bash
python web_app.py
```
ואז פתח בדפדפן: http://localhost:5000

#### 3. שימוש כתוכנית / Programmatic Usage:
```python
from main import KindergartenRecordingAnalyzer

analyzer = KindergartenRecordingAnalyzer()
results = analyzer.run_complete_analysis("recording.wav")
report = analyzer.generate_report(results)
```

### Supported Audio Formats / פורמטי אודיו נתמכים:
- WAV, MP3, M4A, FLAC, AAC, OGG
- עד 500MB לקובץ
- כל קצב דגימה נתמך (מומלץ 22kHz+)

### Files Structure / מבנה קבצים:

#### Core Modules / מודולים עיקריים:
- `main.py` - נקודת כניסה ראשית לאפליקציה
- `audio_analyzer.py` - מודול ניתוח אודיו בסיסי
- `emotion_detector.py` - זיהוי רגשות וטון דיבור
- `cry_detector.py` - זיהוי בכי תינוקות ותגובות
- `violence_detector.py` - זיהוי אלימות ואגרסיה
- `neglect_detector.py` - זיהוי הזנחה וחוסר תשומת לב
- `report_generator.py` - יצירת דוחות מפורטים

#### Web Interface / ממשק ווב:
- `web_app.py` - אפליקציית Flask לממשק ווב
- `templates/index.html` - ממשק משתמש (נוצר אוטומטית)

#### Examples & Documentation / דוגמאות ותיעוד:
- `example_usage.py` - דוגמת שימוש מפורטת
- `requirements.txt` - רשימת תלויות Python
- `README.md` - מדריך זה

#### Generated Directories / תיקיות שנוצרות:
- `uploads/` - קבצי אודיו שהועלו
- `reports/` - דוחות שנוצרו (JSON, HTML, CSV)
- `models/` - מודלים מאומנים (אם ישנם)

### Analysis Output / פלט הניתוח:

המערכת מייצרת דוחות בפורמטים שונים:

#### 1. JSON Report / דוח JSON:
- נתונים מובנים לכל ניתוח נוסף
- כולל כל התוצאות הגולמיות

#### 2. HTML Report / דוח HTML:
- דוח ויזואלי בעברית
- כולל גרפים וסיכומים
- ניתן לצפייה בדפדפן

#### 3. CSV Report / דוח CSV:
- טבלה של כל האירועים
- מתאים לייבוא ל-Excel או כלי ניתוח אחרים

### Key Metrics / מדדים עיקריים:

#### Emotion Analysis / ניתוח רגשי:
- **כעס** - זיהוי כעס בקול הצוות
- **לחץ** - זיהוי מתח וסטרס
- **אגרסיה** - זיהוי התנהגות אגרסיבית
- **רגוע** - זיהוי טון רגוע ונורמלי

#### Cry Detection / זיהוי בכי:
- **זיהוי בכי** - זיהוי אוטומטי של בכי תינוקות
- **עוצמת בכי** - חלש, בינוני, חזק
- **תגובת צוות** - בדיקה אם הצוות הגיב לבכי
- **איכות תגובה** - הערכת איכות התגובה

#### Violence Detection / זיהוי אלימות:
- **צעקות** - זיהוי צעקות ואגרסיה
- **טון אגרסיבי** - זיהוי טון מאיים
- **אלימות פיזית** - סימנים לאלימות פיזית פוטנציאלית
- **הקשר** - ניתוח ההקשר סביב האירועים

#### Neglect Detection / זיהוי הזנחה:
- **בכי ללא תגובה** - זיהוי בכי שלא זכה לתגובה
- **שקט ממושך** - זיהוי תקופות שקט חריגות
- **חוסר אינטראקציה** - זיהוי חוסר תקשורת עם הילדים
- **הזנחת מצוקה** - זיהוי התעלמות ממצוקת ילדים

### Troubleshooting / פתרון בעיות:

#### שגיאות נפוצות / Common Errors:

1. **"ModuleNotFoundError"**
   ```bash
   pip install -r requirements.txt
   ```

2. **"Audio file not found"**
   - וודא שהנתיב לקובץ נכון
   - בדוק שהקובץ קיים וניתן לקריאה

3. **"Out of memory"**
   - נסה קובץ אודיו קצר יותר
   - סגור תוכניות אחרות

4. **"Web interface not loading"**
   - בדוק שהפורט 5000 פנוי
   - נסה: http://127.0.0.1:5000
